# Aplikasi Tes Tipologi Mendalam

Sebuah aplikasi React + TypeScript + Vite untuk mengukur tipologi kepribadian secara mendalam, mencakup:
- MBTI (Fungsi Kognitif)
- Enneagram (Wing, Instinct, Tritype)
- Big Five
- Temperament
- Socionics Clue
- Attitudinal Psyche
- Moral Alignment

Aplikasi ini menggunakan bahasa Indonesia dengan pertanyaan berbasis situasi nyata, bukan sekadar teori.
